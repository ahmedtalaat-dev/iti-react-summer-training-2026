# Day 01 Task

## Description

The objective of this task was to write a professional AI prompt capable of generating a complete React.js learning roadmap in 30 days.

---

## Requirements

The generated output should include:

- A 30-day React.js roadmap.
- Daily learning objectives.
- Daily practical tasks and projects.
- Recommended resources and documentation.
- A PDF document containing the roadmap.
- A responsive website to visualize the learning plan.
