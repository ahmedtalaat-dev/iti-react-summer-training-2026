# Day 01 - Generative AI and Prompt Engineering

## Overview

The first day of the ITI React Summer Training introduced the fundamentals of Generative AI and Prompt Engineering. These concepts are essential for effectively leveraging modern AI tools in software development and learning workflows.

---

## Topics Covered

- What is Generative AI?
- What is an LLM?
- AI Terminologies
- Prompt Engineering
- Prompt Components
- Prompting Techniques
- Prompt Engineering Best Practices

---

## Learning Objectives

By the end of this day, I was able to:

- Understand how Generative AI works.
- Differentiate between common AI concepts and terminologies.
- Learn the fundamentals of Prompt Engineering.
- Explore various prompting techniques and their use cases.
- Apply Prompt Engineering principles to build professional AI prompts.
