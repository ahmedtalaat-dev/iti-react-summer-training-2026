# What is an LLM?

## Introduction

**LLM** stands for **Large Language Model**. It is a type of Artificial Intelligence (AI) model that can understand and generate human-like text.

LLMs are a key technology behind **Generative AI** applications such as chatbots, AI assistants, text generation tools, and code generation systems.

---

## How Does an LLM Work?

LLMs are trained on huge amounts of text data such as books, articles, websites, and code. During training, the model learns patterns, relationships, and structures in language.

When a user provides a prompt, the LLM predicts and generates the most suitable response based on what it has learned.

Example:


Prompt:
```text
Explain machine learning.
```

LLM Response:
```text
Machine learning is a branch of AI that allows computers to learn from data.
```

---

## Key Concepts in LLMs

### Tokens

LLMs process text by breaking it into smaller units called **tokens**. Tokens can be words, parts of words, or characters.

### Parameters

Parameters are the values learned during training that help the model understand language patterns.

### Transformer Architecture

Modern LLMs use the **Transformer architecture**, which helps models understand context and relationships between words.

## What Can LLMs Do?

LLMs can perform tasks such as:

- Text generation
- Question answering
- Summarization
- Translation
- Code generation
- Content creation

---

## LLMs and Prompt Engineering

Users interact with LLMs using **prompts**. Writing clear and effective prompts helps LLMs produce better results.

Example:

```text
Create a summary of Generative AI in 5 bullet points.
```

Good prompt engineering improves accuracy, relevance, and quality of responses.

---

## Limitations of LLMs

- They can generate incorrect information (hallucinations).
- They do not have human-like understanding.
- Their knowledge depends on training data.
- They may produce biased results.
