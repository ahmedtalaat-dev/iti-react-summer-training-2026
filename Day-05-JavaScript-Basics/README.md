# Day 05 - JavaScript Basics

## Overview

The fifth day of the ITI React Summer Training introduced JavaScript, the programming language responsible for adding interactivity and dynamic behavior to web applications.

This session covered the fundamentals of JavaScript, including language types, linking JavaScript files, output methods, events, and variables.

---

## Topics Covered

- Introduction to JavaScript
- Markup Languages vs Scripting Languages vs Programming Languages
- Linking JavaScript to HTML
- JavaScript Output Methods
- JavaScript Events
- JavaScript Variables

---

## Learning Objectives

By the end of this day, I was able to:

- Understand the role of JavaScript in web development.
- Differentiate between markup, scripting, and programming languages.
- Link JavaScript files to HTML documents.
- Display output using different JavaScript methods.
- Handle user interactions through events.
- Declare and use variables.

## Key Takeaways
