- Variables

# Part 02 - Operators

## Requirements

Create two variables:

```javascript

let b;
```

Perform the following operations:

- Addition
- Subtraction
- Multiplication
- Division
- Modulus (Remainder)

Display all results in the console.

### Concepts Practiced

# Part 03 - If Conditions

## Requirements

### Exercise 1

Ask the user to enter a number.

Determine whether the number is:

- Positive
- Negative
- Zero

---

### Exercise 2

Ask the user to enter their age.

Check whether the user is eligible to vote.

Conditions:

```javascript
age >= 18
```

Output:

- Eligible to Vote
- Not Eligible to Vote

### Concepts Practiced

- prompt()
- if Statement
- if...else Statement
- Comparison Operators
