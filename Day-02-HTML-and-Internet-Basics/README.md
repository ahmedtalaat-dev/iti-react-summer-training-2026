# Day 02 - HTML & Internet Basics

## Overview

On Day 02 of the ITI React Summer Training Program, I learned the fundamentals of how the Internet works and started learning HTML.

This day focused on understanding:

- What the Internet is.
- How websites are delivered to users.
- HTML fundamentals.
- HTML tags.
- Images in HTML.
- Meta tags.
- Meta Refresh.
- Lists in HTML.

---

## Learning Objectives

By the end of this day, I was able to:

- Understand the basic concept of the Internet.
- Understand the role of HTML in web development.
- Write basic HTML documents.
- Use common HTML tags.
- Add images to web pages.
- Use meta tags inside the head element.
- Create ordered and unordered lists.
- Use meta refresh for automatic page redirection or refreshing.
