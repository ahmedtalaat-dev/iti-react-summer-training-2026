# Internet Basics

## What is the Internet?

The Internet is a global network of computers that are connected together and communicate using standard protocols.

It allows users to:

- Browse websites.
- Send emails.
- Watch videos.
- Share files.
- Use web applications.

---

## How the Internet Works

When you visit a website:

1. You enter a URL in the browser.
2. The browser sends a request to a server.
3. The server processes the request.
4. The server sends back the website files.
5. The browser displays the webpage.

```text
Browser ---> Request ---> Server

Browser <--- Response <--- Server
```

---

## Main Components of the Internet

- Client (Browser)
- Server
- Internet Connection
- Domain Name
- DNS
- HTTP / HTTPS

---

## What is a Browser?

A browser is a software application used to access websites.

Examples:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari
- Opera

---

## What is a Server?

A server is a computer that stores website files and responds to requests from users.

Examples:

- Web Servers
- Database Servers
- Cloud Servers

---

## What is HTTP?

HTTP stands for:

> HyperText Transfer Protocol

It is the protocol used to transfer data between browsers and web servers.

Example:

```text
http://example.com
```

---

## What is HTTPS?

HTTPS stands for:

> HyperText Transfer Protocol Secure

It provides encrypted and secure communication between the browser and the server.

Example:

```text
https://github.com
```

---

## Summary

- The Internet connects computers worldwide.
- Browsers request data from servers.
- Servers send website files to browsers.
- HTTP and HTTPS are communication protocols used on the web.
