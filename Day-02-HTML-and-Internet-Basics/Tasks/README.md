## Task 02 - Build a Profile Website Using HTML

### Description

The objective of this task is to build a simple personal profile website using only HTML.

This task is designed to practice and apply the HTML concepts learned during Day 02 of the ITI React Summer Training Program.

---

### Requirements

The profile website should include:

- Personal information.
- A profile image.
- Skills, hobbies, or interests displayed using lists.
- A proper HTML document structure.
- Clean and well-organized HTML code.

---

### Concepts Applied

The task covers the following HTML fundamentals:

- HTML document structure.
- HTML tags and elements.
- Images in HTML.
- Meta tags.
- Ordered and unordered lists.

---

### Learning Outcomes

By completing this task, I practiced:

- Building a complete HTML webpage.
- Structuring content using HTML elements.
- Working with images and lists.
- Applying basic HTML best practices.

---

### Task Objective

Build a simple personal profile webpage using HTML only to reinforce the concepts learned during Day 02 of the ITI React Summer Training Program.
