# Day 06 - JavaScript Basics

## Overview

This day focused on learning some of the most important JavaScript fundamentals that are used in almost every application.

Topics covered:

- Loops
- Built-in Functions
- User Defined Functions
- Arrays

---

## Learning Objectives

By the end of this day, I was able to:

- Repeat code using loops.
- Use built-in JavaScript functions.
- Create custom functions.
- Pass data through parameters and arguments.
- Return values from functions.
- Store and manipulate collections of data using arrays.
