# HTML Tables

## What are HTML Tables?

HTML tables are used to display data in rows and columns.

They are commonly used for:

- Student records
- Product lists
- Financial reports
- User information

---

## Basic Table Structure

```html
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>

    <tr>
        <td>Ahmed</td>
        <td>21</td>
    </tr>
</table>
```

## Table Tags

| Tag | Description |
|-----|-----|
| table | Creates a table |
| tr | Creates a table row |
| th | Creates a table heading |
| td | Creates a table cell |
| caption | Adds a title to the table |

---

## Best Practices

- Use table headings whenever possible.
- Keep tables well-structured.
- Use tables only for displaying data, not page layouts.

---

## Key Takeaways

- Tables organize information efficiently.
- They improve readability when presenting data.
