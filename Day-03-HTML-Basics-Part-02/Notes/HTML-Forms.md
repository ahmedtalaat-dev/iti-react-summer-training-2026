# HTML Forms

## What are HTML Forms?

HTML forms are used to collect information from users.

Examples include:

- Registration forms
- Login forms
- Contact forms
- Payment forms

---

## Common Form Elements

- input
- label
- textarea
- select
- option
- button
- fieldset
- legend

---

## Input Types

- text
- email
- password
- number
- date
- radio
- checkbox
- file
- submit
- reset

---

## Example

```html
<form>

<label>Name:</label>
<input type="text">

<label>Email:</label>
<input type="email">

<input type="submit">

</form>
```

---

## Best Practices

- Always use labels.
- Use appropriate input types.
- Group related inputs.
- Make forms accessible and easy to use.

---

## Key Takeaways

- Forms are used to collect user data.
- Different input types provide different functionality.
