# HTML IFrames

## What is an IFrame?

An iframe allows you to embed another webpage or external content inside your own webpage.

Common use cases include:

- Google Maps
- YouTube videos
- External websites
- Interactive content

---

## Syntax

```html
<iframe src="URL"></iframe>
```

---

## Example

```html
<iframe
src="https://www.youtube.com"
width="600"
height="400">
</iframe>
```

---

## Common Attributes

- src
- width
- height
- title
- allowfullscreen
- loading

---

## Best Practices

- Always specify width and height.
- Use descriptive titles for accessibility.
- Only embed trusted content.

---

## Key Takeaways

- IFrames are useful for embedding external resources.
- They improve user experience when used appropriately.
