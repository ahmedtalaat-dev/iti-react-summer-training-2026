# Day 04 - CSS Basics

## Overview

The fourth day of the ITI React Summer Training introduced the fundamentals of CSS (Cascading Style Sheets), which is responsible for styling and designing web pages.

This day covered the essential CSS concepts required to create visually appealing and responsive user interfaces.

---

## Topics Covered

- CSS Fundamentals
- CSS Selectors
- Typography and Colors
- CSS Measurements and Units
- CSS Box Model
- Positioning
- Display Property
- Float and Clear
- Overflow
- Visibility and Opacity
- Z-index
- CSS Best Practices

---

## Learning Objectives

By the end of this day, I was able to:

- Style HTML elements using CSS.
- Understand how CSS rules are applied.
- Use different CSS selectors effectively.
- Work with positioning and layouts.
- Apply typography and color styles.
- Build well-structured and maintainable stylesheets.
